ComfyUI-Integration

Verbindet Photofant mit einer laufenden ComfyUI-Instanz. Photofant triggert Workflows — ComfyUI führt aus.

Verbindung
ComfyUI-Integration aktivieren
Workflow-Trigger sind nur bei aktivierter Verbindung verfügbar.
ComfyUI-URL
Host und Port der ComfyUI-Instanz. Default: http://127.0.0.1:8188
Client-ID
Kennung für die Queue-Zuordnung in ComfyUI. Default: photofant
HTTP-Timeout (s)
Maximale Wartezeit pro Request. Default: 10 s
Output-Ordner (optional)
Lokaler Pfad zu ComfyUIs output/-Ordner — nur für Direkt-Import bei lokalem Betrieb nötig.
Standard-Workflows
Upscale
Wird für den 1-Klick-Hochskalieren-Auftrag aus der Galerie verwendet.
Image Edit
Wird im Editor für promptgesteuertes Umschreiben des Bildes verwendet.
Inpaint
Wird im Editor für das Füllen markierter Bereiche mit einer gemalten Maske verwendet.
