MCP-Schnittstelle

Öffnet Photofant für einen lokalen KI-Agenten (z. B. Claude Desktop). Der Agent kann dann deinen Bestand durchsuchen, ansehen und organisieren — praktisch alles, was diese Oberfläche kann.

Nur mit lokal laufenden Agenten nutzen. Verbindest du einen Cloud-Agenten, landen deine Bilder und Metadaten beim Cloud-Anbieter. Es gibt bewusst keine Passwort-Abfrage — der Schutz ist allein dieser Schalter, und dass die Schnittstelle nur auf diesem Rechner (127.0.0.1) erreichbar ist.
Zugriff
Schnittstelle aktivieren
Standardmäßig aus. Nur einschalten, wenn gerade ein lokaler Agent zugreifen soll.
Rückfrage vor dem endgültigen Löschen
Empfohlen an. Schützt vor versehentlich zerstörerischen Agenten-Aktionen.
Verhalten
Bilder an den Agenten liefern
Aus lassen, wenn du dem Agenten die Bildinhalte nicht zeigen willst.
Treffer pro Suche (max.)
1–500. Der Agent kann bei Bedarf weiterblättern.
Vorschaugröße (px)
64–1024 px. Größer heißt schärfer, aber deutlich mehr Datenverbrauch.
